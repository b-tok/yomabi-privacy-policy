# Yomabi Privacy Policy

This repository hosts the privacy policy for [Yomabi](https://play.google.com/store/apps/details?id=com.yomabi), a Japanese dictionary and learning app for Android.

The privacy policy is published via GitHub Pages at the repository's root `index.html`.

## Updating

Edit `index.html` directly. The page has no external dependencies -- all styling is self-contained. It supports both light and dark modes via `prefers-color-scheme`.
